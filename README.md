# Ang How Cheng Wilson - Aerospace Portfolio

A personal portfolio website showcasing my journey, achievements, and aspirations in aerospace engineering.

## 🚀 Website

**Live at:** [https://wilsonang2010-boop.github.io/eae-portfolio/](https://wilsonang2010-boop.github.io/eae-portfolio/)

## About

This portfolio highlights my:
- Academic journey and milestones
- Experience in aerospace and leadership
- Achievements and certifications
- Flight mechanics and aviation knowledge
- Passion for aerospace engineering

## Features

- Clean, minimal design inspired by Swiss and Apple aesthetics
- Fully responsive (mobile, tablet, desktop)
- Image gallery with lightbox navigation
- Timeline of key milestones
- Testimonials from mentors and educators
- Contact information for Singapore Polytechnic admissions
